# YAML Basics

**Date:** 2026-06-28
**Topic:** YAML Syntax

## What was learned
The user learned the three golden rules of YAML:
1. Key-value pairs (with a space after the colon).
2. Indentation matters (using spaces, not tabs) to denote nesting.
3. Lists use dashes (with a space after the dash).

## Observations
The user recognized that spacing/indentation is a common pitfall in YAML files. They were able to identify spacing issues in a buggy YAML snippet.
